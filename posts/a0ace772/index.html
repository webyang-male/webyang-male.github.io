PicGo搭建Gitee图床

2021-05-16
837- 3m
其他-其他

准备环境

  • PicGo
  • Gitee

环境简介

Gitee是开源中国（OSChina）推出的基于Git的代码托管服务

其实和GitHub一样，一个国内一个国外，另外Coding也是国内较大的代码托管服务商

PicGo: 一个用于快速上传图片并获取图片 URL 链接的工具

PicGo 本体支持如下图床：

  • 七牛图床 v1.0
  • 腾讯云 COS v4\v5 版本 v1.1 & v1.5.0
  • 又拍云 v1.2.0
  • GitHub v1.5.0
  • SM.MS V2 v2.3.0-beta.0
  • 阿里云 OSS v1.6.0
  • Imgur v1.6.0

🌟TIPS:

​ 可以通过安装插件增强PicGo功能并支持搭建Gitee图床

官方文档

https://picgo.github.io/PicGo-Doc/zh/guide/

下载地址

https://github.com/Molunerfinn/PicGo/releases

搭建步骤

下载安装

Windows 下载最新版本的 exe 文件即可。

应用截图

下载插件

点击插件设置,在输入框输入gitee关键字，将出现下图结果：

你可以先点开左侧图床设置，默认支持图床没有Gitee。

这里因为我安装过了,所以显示已经安装,你可以选择和我一样的插件。

加速下载

因为插件服务在国外,可以选择更改PicGo的代理和镜像源，在PicGo设置中可以查看

https://registry.npm.taobao.org/
Gitee配置

  • 新建一个仓库(此步骤略)

  • 新建 私人令牌
    点击头像 —— 设置 —— 私人令牌 —— 复制令牌密匙

    这个令牌很重要,不要泄露遗失,后面配置需要用到(虽然丢了可以删除再重新弄一个)

    分配权限勾选前3个其实已经够用了

  • 在PicGo侧边功能区图床设置中选择gitee，开始输入参数配置

配置参数说明：

  • repo: gitee用户名/仓库名；
  • branch: 分支名（如默认：master）
  • token: gitee私人令牌
  • path: 用于仓库下存储的目录
  • customPath和customUrl可以不用填

简约方式:填写图中前3项就足够了!!!

坑点

⚠️不能上传图片很有可能就是repo项填写错误,这里内容不一定是是gitee页面显示的xxxx / xxxx,而是在仓库页面浏览器url的https://gitee.com/后面所显示的路径名,直接复制粘贴进就行

✍答应我，建议仓库名不要奇奇怪怪带空格或者符号好嘛?

上传测试

在PicGo侧边功能区上传区上传图片,可选直接拖动、点击本地上传、剪贴板粘贴，甚至URL上传（左边可选URL的格式）

记得在大大的图片上传旁边选好你想上传的图床gitee，如下图：

上传成功与否可以看是否一条蓝线快速划过（笑

准确做法是看左侧功能区相册是否有图片展示，然后康康你的gitee存放图片的仓库有木有图片上传

结语

最坑的也就是PicGo配置gitee的repo变量，**的原来是用户名，不是你的昵称，除了国内访问友好，还不如github所见即所得。
